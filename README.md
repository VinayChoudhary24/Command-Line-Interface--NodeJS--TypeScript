Usage: index [options]

An example CLI for managing a directory

Options:
  -V, --version        output the version number
  
  -l, --ls  [value]    List directory contents
  
  -m, --mkdir <value>  Create a directory
  
  -t, --touch <value>  Create a file
  
  -h, --help           display help for command
